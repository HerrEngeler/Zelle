Datenschutz auf einen Blick

1. Verantwortliche Stelle

Jonas Engeler (siehe Impressum). Dieses Projekt ist nicht-kommerziell und wird als Bildungsprojekt betrieben.

2. Erhobene Daten

Diese Website speichert keine personenbezogenen Daten auf einem Server. Es findet keine Übertragung von Nutzerdaten statt.

localStorage: Der Lernfortschritt (welche Labore abgeschlossen wurden) wird ausschließlich im localStorage des eigenen Browsers gespeichert. Schlüssel: zellen-escape-room_progress_v1. Diese Daten verlassen das Gerät nicht und können jederzeit über die Fortschritt-Zurücksetzen-Funktion oder die Browser-Einstellungen gelöscht werden.

sessionStorage: Temporäre Daten wie Musik-Einstellungen und Lehrer-Login werden im sessionStorage gespeichert. Diese Daten werden automatisch gelöscht, wenn der Browser-Tab geschlossen wird.

3. Google Fonts

Diese Website verwendet Google Fonts (Playfair Display, Lato, Courier Prime). Beim Laden der Schriftarten wird eine Verbindung zu Googles Servern hergestellt, wobei die IP-Adresse des Nutzers übertragen werden kann. Weitere Informationen: Google Datenschutzerklärung.

4. GitHub Pages (Hosting)

Wenn diese Website über GitHub Pages bereitgestellt wird, können dort Server-Logdaten (IP-Adresse, Zugriffszeit, aufgerufene URL) für maximal 14 Tage gespeichert werden. Weitere Informationen: GitHub Datenschutzerklärung.

5. Ihre Rechte

Da keine personenbezogenen Daten auf unseren Servern verarbeitet werden, entfallen die meisten DSGVO-Rechte auf Auskunft, Berichtigung oder Löschung. Den lokal gespeicherten Fortschritt können Sie jederzeit selbst löschen (Browser-Einstellungen → Websitedaten löschen oder über die Zurücksetzen-Funktion auf der Labor-Übersicht).
