ℹ️ Hinweise zur Nutzung

Dieser Escape Room ist eine statische Website – es werden keine Daten an Server übertragen. Der Lernfortschritt wird ausschließlich im localStorage des Browsers gespeichert.

Für den Unterricht empfiehlt sich der Einsatz auf Schülergeräten (Tablets, Laptops) in Einzel- oder Partnerarbeit. Der Lehrmodus ermöglicht Demo-Vorführungen auf dem Lehrerrechner.
